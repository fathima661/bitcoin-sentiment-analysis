# Bitcoin Sentiment vs Trader Performance Analysis

This project analyzes the relationship between Bitcoin market sentiment (Fear & Greed Index) and trader performance using Hyperliquid data.

## Key Insights
- Highest profits occur during Extreme Greed
- Traders take higher risks during Fear phases
- Trading activity remains balanced across sentiments
- A small group of traders dominates profits

## Conclusion
Effective strategies involve buying during Fear and managing risk during Greed phases.
